# CampusVoice - Multi College Complaint Portal

A web application that allows students to submit and track campus complaints, and administrators to manage and resolve them efficiently.

## Features
- Multi-college support
- Student & Admin roles
- Complaint tracking with status updates
- Priority levels (Low/Medium/High/Urgent)
- Admin reply system
- Charts and analytics

## Tech Stack
- React.js
- Tailwind CSS
- Next.js

## Getting Started
npm run dev
